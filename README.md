# Geometry Formulas Library

Это библиотека для вычисления площадей и периметров геометрических фигур.

## Фигуры и функции:

### Круг:
- area(r) - площадь круга: π * r²
- perimeter(r) - длина окружности: 2 * π * r

### Прямоугольник:
- area(a, b) - площадь прямоугольника: a * b  
- perimeter(a, b) - периметр прямоугольника: 2 * (a + b)

### Квадрат:
- area(a) - площадь квадрата: a²
- perimeter(a) - периметр квадрата: 4 * a

### Треугольник:
- area(a, h) - площадь треугольника: (a * h) / 2
- perimeter(a, b, c) - периметр треугольника: a + b + c

## Использование:
import math
from geometry import *

# Примеры:     
# Площадь круга радиусом 5
- circle_area = area(5)
# Периметр прямоугольника 4x6
- rect_perimeter = perimeter(4, 6)
